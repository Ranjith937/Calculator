## Calculator
A responsive calculator web application built using HTML, CSS, and JavaScript. The app supports basic arithmetic operations, real-time expression display, clear and delete functions, and a modern calculator interface built with CSS Grid. It provides a simple and interactive way to perform calculations in the browser.

## 🧮 Calculator Web Application

A simple and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.  
The application allows users to perform basic arithmetic operations through an intuitive and modern calculator interface.

---

## 🚀 Features

- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
  - Modulus (%)

- Real-time expression display
- Clear all calculations using **AC**
- Delete the last entered value
- Prevents invalid operator sequences
- Responsive and modern calculator UI
- Interactive button effects

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the calculator
- **CSS3** – Styling and layout using CSS Grid
- **JavaScript (Vanilla JS)** – Handling calculations and user interactions
